# AI Hairstyle Recommender - Frontend
This is the React-based user interface for the AI Hairstyle Recommender. It allows users to upload a portrait photo and receive real-time hairstyle suggestions based on computer vision analysis of their facial geometry.

## Tech Stack
* **React**: Component-based UI.
* **Tailwind CSS**: Modern, responsive styling.
* **Axios**: API communication with the FastAPI backend.

## Getting Started
1. Install dependencies: `npm install`
2. Start the development server: `npm start`
3. The app connects to the backend at `http://localhost:8000`.

## Features
* Secure image upload and preview.
* Real-time display of face shape classification (Oval, Square, Heart, etc.).
* Dynamic recommendation list based on professional barbering logic.
